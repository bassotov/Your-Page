# Pasha Barbashin - Personal Page

Personal portfolio and link hub built with Next.js, Tailwind CSS, and shadcn/ui.

## Live Site

[barbash.in](https://barbash.in)

## Features

- Responsive design with light/dark theme support
- 2025 Wrapped section showcasing yearly stats from various platforms
- Projects showcase with image cards
- Social links footer

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Components**: shadcn/ui
- **Icons**: Lucide React
- **Theming**: next-themes with OKLCH colors

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```
